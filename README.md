# AWS-DynamoDB-CRUD


Steps to setup:
1. AWS Java SDK
2. AWS CLI MSI Installer Interface.
3. GET access key and secret key.
4. configure aws default profile
   command: >aws configure
5. external jars:
-common-logging
-httpclient
-httpcore
-jackson-databind
-com.fasterxml.jackson.databind
-jackson-annotation
-jackson-core
-joda-time
